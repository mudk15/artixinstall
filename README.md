# artixinstall
## git
1. Enter livecd
2. Enter root (su)
3. pacman -Sy git
4. git clone https://github.com/mudk15/artixinstall
5. cd artixinstall
6. ./script.sh
## wget 
1. Enter livecd
2. Enter root (su)
3. pacman -Sy wget
4. wget https://raw.githubusercontent.com/mudk15/artixinstall/main/script.sh
5. chmod 777 script.sh
6. ./script.sh
